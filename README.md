# Game Edukasi

Dokumentasi proyek Laravel untuk aplikasi edukasi interaktif dengan fitur artikel, video, dan game room berbasis kartu pertanyaan.

## Dokumentasi

- [Ringkasan Proyek](./docs/overview.md)
- [Spesifikasi Sistem](./docs/system-specification.md)
- [Roadmap Implementasi](./docs/implementation-roadmap.md)

## Stack

- Laravel
- Blade
- Tailwind CSS
- jQuery
- SweetAlert2
- DataTables
- MySQL
- Laravel Mail
- Laravel Storage

## Catatan Arsitektur

- Realtime chat dan gameplay tahap awal menggunakan AJAX polling.
- Struktur sistem disiapkan agar mudah di-upgrade ke Laravel Reverb atau Pusher.
- Layout admin dan user dipisah agar pengalaman penggunaan lebih jelas.

## Status

Saat ini repo difokuskan sebagai dasar proyek dan dokumentasi kebutuhan sistem.
